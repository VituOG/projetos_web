lang="pt-br"> Lista

Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes em uma receita

A tag li (list itens - itens da lista) representa um item que faz parte de uma lista.

Exemplo de lista

  • primeiro item
  • segundo item
  • terceiro item
    • sub-item

Listas ordenadas

A tag ol (ordened list- lista ordenada) representa uma lista de itens numerados sequencialmente

Exemplo de lista ordenada

  1. Primeiro item
  2. Segundo item

Atributos

reversed (ordem reversa)
exibe os itens da lista ao contrario

  1. item mais importante
  2. item menos importante

start(inicio)
especifica o valor inicial para a numeração da lista.
