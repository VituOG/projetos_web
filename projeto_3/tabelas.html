o elemento table (tabela)

o elemento table (tabela) representa dados tubulares ou seja, informações que são representados em linhas e coluna

Exemplo

Tabela simples com cabeçalho

primeiro nome sobrenome
vitor carvalho
stro 200

Elementos da tabela

  • table - tabela
  • th (table header) - cabeçalho da coluna
  • tr (table row) - linha da tabela
  • td (table data) - valor/conteudo da celula
